# SIPEKA - Sistem Informasi Pelaporan Kinerja

Aplikasi berbasis web untuk memudahkan pelaporan kinerja harian dan monitoring rencana kerja secara mandiri.

## 🚀 Fitur Utama
- **Login Sistem**: Keamanan akses menggunakan NIK dan Password.
- **Rencana Kerja (Renja)**: Input dan manajemen rencana kerja bulanan.
- **Laporan Harian**: Pelaporan realisasi kegiatan dilengkapi dengan unggah foto visum.
- **Dashboard Grafik**: Visualisasi pencapaian kinerja dalam bentuk grafik bar.
- **Manajemen User**: Fitur khusus admin untuk aktivasi/non-aktivasi akun kader.
- **Cetak Visum**: Fitur ekspor laporan ke format PDF siap cetak.

## 🛠️ Teknologi yang Digunakan
- **Frontend**: HTML5, CSS3 (Tailwind CSS), JavaScript (Vanilla).
- **Backend**: Google Apps Script (GAS).
- **Database**: Google Sheets.
- **Hosting**: Vercel / Netlify.
- **Library**: Chart.js (untuk grafik).

## 📋 Cara Penggunaan
1. Akses link aplikasi yang sudah di-deploy.
2. Login menggunakan NIK yang sudah terdaftar.
3. Gunakan menu **Renja** untuk membuat rencana kerja terlebih dahulu.
4. Gunakan menu **Laporan** untuk melaporkan kegiatan yang sudah selesai.
5. Admin dapat masuk ke menu **Approve** untuk melakukan verifikasi laporan.
